Description and Concept:

This project is to create an imaginary creature using the p5 editor. I created some cute dragons, which contain transformation functions and push () & pop () in order to make the rotation of their teeth and tails. Also, it contains an array to change the size of the dragons. Different functions are also included to create different parts of the dragon.

Coding Progress:

For my project, the most important part is the function drawCreature (). Since it contains all the main parts of a dragon and calls all the functions of different body parts of a dragon. At first, I tried to make all three of them have color-changing teeth, however, I found out that the positions of the teeth are uncertain, I bet it’s the rotation part that went wrong, and I failed to correct them. Reflection

-I used numbers stored in the array, in order to change the size of the dragons. I wanted to create random size dragons, however, their size will change all the time if I code random(). I may need further exploration of array ().

-We need to use translate () when moving the center position, in order to simplify the process of rotating a shape around its center point. Push () and pop () are used to make the codes between them independent from the codes outside. Therefore, we can change a special part of our code while not affecting other parts.

-In this case, we can only create one function and call it several times to create a number of shapes with different measures.

-Position, rotation, and size. I can create dragons at different positions, rotate their tails at different rates, and also create dragons with different sizes.

-I can no longer write so many codes to create a similar pattern, but just write a function and call it several times. I think it’s convenient and practical.
